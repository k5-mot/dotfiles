# dotfiles

## Features

### Toolsets

* bash
* emacs
* git
* latexmk
* tmux
* vim
* neovim
* zsh

### Scripts

* anyenv
  * pyenv
  * rbenv
  * nodenv
  * luaenv
  * plenv
* apt
* porg

## Requirement

* Vim 8.1
* Neovim 0.4.3
* Emacs 26.3

## Installation

* Case 1<br>

```zsh
git clone https://github.com/k5-mot/dotfiles.git $HOME
cd dotfiles
./install.sh
```

* Case 2<br>
```zsh
curl -L https://raw.githubusercontent.com/k5-mot/dotfiles/master/bin/install.sh | bash
```
