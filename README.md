# dotfiles

## Features
 
Start up for various environment.
 
## Requirement
 
* Vim 8.1
* Neovim 0.4.3
* Emacs 26.3

## Installation
 
* Case 1<br>

```bash
git clone https://github.com/k5-mot/dotfiles.git $HOME
cd dotfiles
./install.sh
```

* Case 2<br>
WIP
```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/k5-mot/dotfiles/master/bin/install.sh)"
```
