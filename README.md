# dotfiles

## Features
 
Start up for various environment for Linux.

* bash
* emacs
* git
* latexmk
* tcsh
* tmux
* vim
* neovim
* zsh

Start up for various environment for Windows.

* PowerShell
* Windows Terminal

Set up for any applications.

* pyenv
* rbenv
* nodenv

Others' utility

* apt
 
## Requirement
 
* Vim 8.1
* Neovim 0.4.3
* Emacs 26.3

## Installation
 
* Case 1<br>

```bash
git clone https://github.com/k5-mot/dotfiles.git $HOME
cd dotfiles
./install.sh
```

* Case 2<br>
```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/k5-mot/dotfiles/master/bin/install.sh)"
```
