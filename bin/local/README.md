* install_apt_package.sh
 
    * Change Acquisition server to mirror servers in Japan.
    * Install applications.
 
* update_apt.sh

    * Update & upgrade packages.

* install_pyenv.sh

    * Install pyenv.
    * Install plugins in pyenv.
    * Install packages in pip.

* install_rbenv.sh

    * Install rbenv.
    * Install plugins in rbenv.
    * Install packages in gem. 

* install_nodenv.sh

    * Install nodenv.
    * Install plugins in nodenv.
    * Install packages in npm. 
 
## Requirement
